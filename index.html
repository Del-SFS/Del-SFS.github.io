SCAN

SFS Comprehensive Archive Network

About SCAN

SCAN (SFS Comprehensive Archive Network) is an automated mod manager designed for the PC version of Spaceflight Simulator. It replaces manual mod installation with a structured and reliable system.

Instead of manually copying files into game directories, SCAN reads structured metadata and performs installation automatically. This eliminates common user errors and ensures that mods are always placed correctly.

User Interface

The SCAN interface is designed to be simple and functional. Users can manage mods, view available content, and control installation without interacting with game files directly.

How the System Works

SCAN connects to a structured repository containing mod metadata. Each mod is described using a standardized format that includes version, type, dependencies, and conflicts.

The application reads this data and determines how the mod should be installed. Files are automatically mapped to the correct directories based on their type.

  • Automatic file placement
  • Dependency resolution
  • Conflict detection
  • Consistent installation process

Mod Format

All mods follow a strict structure to ensure compatibility with SCAN. 

ModName.zip
 ├── ModName.zip
 └── mod.json

The internal ZIP must contain only mod files such as DLLs, assets, or textures. It must not include unnecessary folders.

Why SCAN

Manual mod installation is error-prone and inconsistent. Different mods require different file placements, and users often make mistakes.

SCAN solves this by standardizing the entire process and automating installation. This creates a reliable environment for both users and mod creators.

  • Reduces installation errors
  • Improves compatibility
  • Simplifies mod management
  • Creates a scalable ecosystem

Ecosystem

SCAN is part of a larger system designed to improve mod distribution.

  • SCAN – Mod installer
  • ModCenterSFS – Mod repository
  • GMJS – Mod creation tool

Together, these components form a complete pipeline for creating, distributing, and installing mods.

Installation

Download SCAN using the button above and run the executable. The program will prepare the environment and allow you to install mods automatically.

No manual configuration or file management is required.
